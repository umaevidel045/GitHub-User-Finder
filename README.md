## Описание программы

** GitHub User Finder ** - это GUI-приложение которое осуществляет поиск пользователей c  GitHub  API и с возможность добавления в избранное (JSON).

### Возможности:
- ✅ Поиск пользователей
- ✅ Загрузка и выгрузка с файла JSON
- ✅ Выдача результатов данных пользователей в таблицу
- ✅ Добавление и удаление избранных пользователей.

## Инструкция по запуску

### Требования:
- Python 3.6 или выше
- Установленная библиотека Tkinter (обычно идёт в комплекте с Python)
- API зависим от requests
- обработка ошибок 

### Запуск:
1. Откройте терминал или командную строку
2. Перейдите в директорию проекта:
3. Открыть файл  app .py
4.В терминале прописать команду pip install requests
5.Запустить приложение python app.py
   
## Примеры использования

### Поиск пользователей
1.Введите имя пользователя 
2. Нажмите кнопку **"Найти"**
3. Результаты появятся в таблице

### Добавление в избранное
- Выберите в таблице нужного пользователя и нажмите кнопку «Добавить в избранное»
- Два раза кликните по таблицу с избранным для удаления пользователя
- Кнопка обновить обновляет список избранного
